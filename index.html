Introduction

RO-SHAM-BO (Rock, Paper, Scissors) has long been the dominate school yard arbitor of debate, worldwide. Two oponents face off, each throwing one of three possible objects: ROCK , PAPER, or SCISSORS. Given the game's popularity and simplicity, Ro-Sham-Bo makes the beginner level JavaScript project.

This game will be played between the user against the computer. The code can be broken down into four parts

  1. User Choice
  2. Computer Choice
  3. User vs. Computer
  4. Run the Program
User Choice

First, the user needs to be able to select between ROCK, PAPER, or SCISSORS. This calls for a function.

  • Create a function: getUserChoice
  • The function should take one parameter: userInput

                //User Choice
                const getUserChoice = userInput => {}
            

                //Can also be written as:
                const getUserChoice = function getUserChoice(userInput){};

Next, it is important to exercise a level of control over the user's input.

  • Use the .toLowerCase() Method
  • This avoids complications if the user decides to get creative with their input
  • Set userInput = userInput.toLowerCase();

                //User Choice
                const getUserChoice = userInput => {
                    userInput = userInput.toLowerCase();    
                }

Now, test the user's input to ensure they made a vailid object section

  • Write an if/else statement ensuring the user entered 'rock', 'paper', or 'scissors'.
  • If the userInput passes the test return the userInput
  • Else console.log an error
  • Test the function using console.log

                //User Choice
                const getUserChoice = userInput => {
                    userInput = userInput.toLowerCase();
                    if (userInput === 'rock' || 'paper' || 'scissors'){
                        return userInput;
                    } else {
                         console.log('ERROR');
                    }
                }

                //TEST User Choice Function
                console.log(getUserChoice('ROCK')); //returns : rock
